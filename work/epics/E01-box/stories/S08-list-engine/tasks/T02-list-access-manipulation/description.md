# T02: List Access and Manipulation

**Status:** done

Implement LRANGE, LINDEX, LSET, LINSERT, LREM, LTRIM, LPOS. LPOS supports RANK, COUNT, MAXLEN options (Redis 6.0.6+). LINSERT inserts BEFORE|AFTER pivot element. LREM removes count occurrences (positive=head-to-tail, negative=tail-to-head, 0=all).

## Acceptance Criteria

- All commands match Redis behavior
- LPOS options work correctly
