# S15: Bitmap, HyperLogLog, and Geo Commands

Implement three specialized command groups that build on existing types. Bitmaps operate on strings, Geo uses sorted sets with geohash encoding, HyperLogLog uses a special string encoding.

- T01: Bitmap commands
- T02: HyperLogLog commands
