# T02: ACL Commands

Implement ACL SETUSER, ACL DELUSER, ACL GETUSER, ACL LIST, ACL WHOAMI, ACL CAT [category], ACL LOG [count|RESET], ACL GENPASS [bits], ACL LOAD (stub), ACL SAVE (stub), ACL DRYRUN user command [arg ...]. ACL CAT returns list of command categories. ACL WHOAMI returns current username.

## Acceptance Criteria

- User management works
- ACL WHOAMI returns correct user
- ACL CAT returns all categories
