# T05: Pop, Random, and Aggregate Operations

Implement ZPOPMIN, ZPOPMAX, ZMPOP, ZRANDMEMBER, ZUNION, ZINTER, ZDIFF, ZUNIONSTORE, ZINTERSTORE, ZDIFFSTORE, ZINTERCARD, ZSCAN. ZMPOP (Redis 7.0+): pop min/max from first non-empty set. ZRANDMEMBER supports WITHSCORES. Aggregate operations support WEIGHTS and AGGREGATE (SUM|MIN|MAX).

## Acceptance Criteria

- All commands match Redis behavior
- Aggregate weights/functions correct
