# T05: PUBSUB Introspection and Sharded Pub/Sub

Implement PUBSUB CHANNELS [pattern], PUBSUB NUMSUB [channel ...], PUBSUB NUMPAT, PUBSUB SHARDCHANNELS, PUBSUB SHARDNUMSUB. Implement SSUBSCRIBE, SUNSUBSCRIBE, SPUBLISH (sharded pub/sub — in non-cluster mode, behaves identically to regular pub/sub).

## Acceptance Criteria

- All PUBSUB subcommands return correct data
- Sharded commands work
